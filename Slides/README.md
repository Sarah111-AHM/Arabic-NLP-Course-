# Slides - Arabic NLP Course

This folder contains all the presentation slides for the **Arabic NLP Course**, organized in a logical progression from fundamentals to advanced topics.

---

## Contents

- **Intro**  
  Course overview and introduction.

- **01 - Pytorch Fundamental**  
  Basics of PyTorch and tensors.

- **02 - Activation Functions and Optimization**  
  Key concepts in training neural networks.

- **03 - Bag of Words and Vector Representations**  
  Classical NLP techniques for text representation.

- **04 - RNNs**  
  Sequence modeling using recurrent networks.

- **05 - Attention Mechanisms**  
  Understanding attention in deep learning.

- **06 - Transformers**  
  Modern NLP architectures.

- **07 - Tokenization**  
  Text preprocessing techniques.

- **08 - RAG**  
  Retrieval-Augmented Generation.

- **09 - Reinforcement Learning**  
  Introduction to RL concepts in NLP.

---

## Structure

Each folder includes slides related to its topic.  
The order reflects a recommended learning path.
